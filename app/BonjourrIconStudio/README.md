# Bonjourr Icon Studio

Нативная портативная утилита для Windows 10/11, которая подготавливает иконки для Bonjourr и публикует их в GitHub Pages.

## Возможности

- импорт PNG, JPEG, WebP, AVIF, BMP и TIFF;
- ручное позиционирование и масштабирование изображения;
- фиксированная суперэллиптическая маска `|x/a|^4.2 + |y/b|^4.2 = 1`;
- отключаемый многослойный контур Liquid Glass в вариантах Light и Dark;
- регулировка толщины контура от 0,5 до 16,0 px с шагом 0,1;
- экспорт 128, 256 и 512 px;
- PNG, WebP и AVIF;
- восьмикратный рендеринг маски и уменьшение Lanczos для гладкого альфа-края;
- загрузка готовых файлов в `onedalist/bonjourr-icons` через GitHub API;
- автоматическое создание ссылки GitHub Pages;
- защищённое сохранение токена через Windows DPAPI;
- переносимый зашифрованный профиль AES-256-GCM + Argon2id.

## Запуск готовой версии

1. Распаковать архив в постоянную папку, например `D:\Soft\Bonjourr Icon Studio`.
2. Запустить `BonjourrIconStudio.exe` двойным щелчком.
3. На странице «Настройки» вставить новый fine-grained GitHub token.
4. При сохранении задать восстановительный пароль.

Установка .NET, открытие терминала и запуск локального сервера не требуются.

## Токен GitHub

Токену достаточно доступа только к репозиторию `onedalist/bonjourr-icons` и разрешения `Contents: Read and write`.

На текущей Windows ключ профиля защищается DPAPI и разблокируется автоматически. Для переноса или восстановления после переустановки используется отдельный пароль. Резервную копию `.bisp` нужно хранить отдельно от системного диска.

## Данные портативной версии

Приложение создаёт рядом с собой:

```text
Data/
  settings.json
  github-profile.enc
Exports/
```

Настройки контура сохраняются в `Data/settings.json`.
Папку приложения можно целиком хранить на несистемном диске.
